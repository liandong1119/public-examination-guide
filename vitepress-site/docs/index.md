### 代码示例
```javascript
// 计算等差数列第n项
function arithmeticSequence(a1, d, n) {
  return a1 + (n - 1) * d;
}

// 使用示例
console.log(arithmeticSequence(2, 3, 5)); // 输出: 14
```
